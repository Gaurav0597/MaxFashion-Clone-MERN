# MaxFashion-Clone-An e-commerce website that deals with Clothing products.

## Technologies :

• ReactJS

• Redux

• NodeJS

• ExpressJS

• MongoDB

## Dependencies :

### Frontend:

• redux

• react-redux

• redux-thunk

• react-router-dom

• axios

• font-awesome

### Backend:

• express

• mongoose

• cors

• dotenv

• mongoose

## Features :
• Sign-in/Sign-up
• Products carousel, sort, and filter
• User authentication
• Add to cart
• Checkout pages
